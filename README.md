# Sistem-Navigasi-Penyelamatan-Korban-Gempa-di-Area-Reruntuhan-Menggunakan-Algoritma-BFS
Implementasi algoritma Breadth First Search (BFS) pada sistem navigasi penyelamatan korban gempa. Proyek ini bertujuan membantu proses evakuasi dengan menentukan urutan kunjungan lokasi secara terstruktur sehingga seluruh area terdampak dapat diperiksa secara menyeluruh.
